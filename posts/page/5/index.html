How to change the default web port on unRAID

You can easily change the default web port for unRAID from port 80 to something you like by changing the “go” file in “/boot/config/”. In the example above I changed my unRAID to listen on “8008” instead of the default “80”. You will have to restart your unRAID server to apply the change.

Deluge Error. No module named gobject

No module named gobject I just installed Deluge on Archlinux but the GUI kept failing to come up with “No module named gobject”. This happens because installing just the deluge package does not meet all the dependencies required to run the GUI. Install “python-gobject2” and “pygtk” packages to satisfy the dependencies and Deluge will run fine. $pacman -Sy python-gobject2 pygtk
