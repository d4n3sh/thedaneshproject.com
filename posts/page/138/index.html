Arkayne related posts plugin for WordPress

I’m trying out a new related posts plugin called “Arkayne” for a few weeks on my blog. I’ve been looking around for related posts plugins and been testing a few recently but most of them either only use tags or categories to find related posts. So far my favorite has been Related Entrie by WASABI because it uses keywords instead. What is Arkayne? Arkayne is an online service that collects and automatically links similar content from the Arkayne network. The Arkayne network is made up of sites/blogs which are connected via the Arkayne plugins or java scripts. Their posts get registered by Arakyne and the magic happens. The Arkayne engine consists of a set of rules and algorithm the relates pages based on content and not just tags, keywords or categories. If Arkayne functions as advertised, someone surfing a PHP related site should only see posts related to the PHP world. ...
